# インターネット電話
linux, mac環境において同一のwifiに繋ぐことでインターネット電話として動作します。
- sox

が入っていれば、動くと思います。
## 注意
- まだpc2台を使って試せていないのできちんと動作するかは分かりません。
## ビルド
```
  gcc -o phone phone.c 
```
## 使い方
- IPアドレスは
```
  $ ifconfig
```
で調べることができます。
- ポート番号は10000以上で設定してください。

### クライアント
```
  ./phone <IPアドレス> <ポート番号>
```
### ホスト
```
  ./phone <ポート番号>
```
